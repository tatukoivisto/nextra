# Nextra Features

Nextra boasts robust support for [MDX](/features/mdx), allowing Markdown and
React integration. It leverages [Next.js SSG](/features/ssg) for optimal
performance, offers [Next.js i18n](/features/i18n) for global reach, and
optimizes visuals with [Next.js Image](/features/image). With customizable
[Themes](/features/themes) and precise [LaTeX](/features/latex) presentation,
it's a comprehensive solution.
